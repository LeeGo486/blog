ES6 shen me gui?

let and const

特点：

  • let 和 const, 都支持块级作用域
  • const 用来声明常量，不能改变
  • 不可重复声明

总结：

  • 能用 const 就用 const, 不能用 const 了再用 let, 坚决不用 var
  • 出乎意料的事：偶尔才会用一下 let

arrow function

特点：

  • 不会改变 this 指向
  • 没有 arguments 对象

总结：

  • `var self = this`, 见鬼去吧

template string

特点：

  • 变量替换
  • 字符串换行

short

特点：

  • Just short

io.js or babel

io.js

支持：

  •  2.0.0 后基本支持了所有 es6 的 features, 待调查。

优点：

  • 语言层面支持，稳定性 + 安全性

缺点：

  • io.js 不够普及，无法保证调用者使用的版本

babel(6to5)

支持：

  •  能用到的都已经支持

优点：

  • 前后端通吃
  • gulp, nodemon, pm2 等都已支持

缺点：

  • 前端：实际运行的代码是编译后的，不利于调试
  • Nodejs: 需要使用 babel 自带的 babel-node 来运行，稳定性无法保证

front: gulp-babel

Nodejs + babel

  node_modules/.bin/babel-node

  nodemon --exec ./node_modules/.bin/babel-node app.js

  ./node_modules/.bin/mocha --compilers js:babel/register

  NODE_ENV=production pm2 start app.js --next-gen-js
